Info

  • Left click to reveal a tile.
  • Right click to mark a tile as a mine. (long press on mobile)
  • Numbers indicate how many adjacent tile are mines.
  • Double-click a number to reveal all (non-marked) adjacent tile.
  • Right-click a number to mark all adjacent tile as mines.
  • Anyone can join a room with you.
  • There are several modes. Drop down the options while in a room to select:
    • Cruel: If you guessed, you guessed wrong. If you had to guess, you guessed correctly. This is the one you should use.
    • Fair: If you must guess, you guessed correctly. No extra punishment for guessing when you didn't have to.
    • Nice: If you guessed, you were correct. Unless it was a really stupid guess.
    • Normal: The mines won't move. Lame.
  • Undo Is there anyone else watching? Are they judging you? Don't worry, the mines will move, you still have to use logic.
  • Knights Move adjacency mode. For sadists.
  • Oh yeah, reveal all the tiles that aren't mines.
  • Source code is on GitHub.
